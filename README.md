# Youtube-Video-Thumbnail

A youtube video link or id to video thumbnail python package.

## Installation

```
pip3 install ytthumb
```

## Usage

```python
import ytthumb

ytthumb.thumbnail("https://youtu.be/rokGy0huYEA")
```
